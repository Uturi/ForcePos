# 키트리 디컨 29기 1차 프로젝트 'ForcePos'

## 1. 개요
계정관리, 상품 판매, 재고 관리, 정산, 통계 기능이 있는 편의점 포스기 프로그램 'ForcePos'입니다.

## 2. 개발환경
- OS : window 7 <br/>
- 개발언어 : java <br/>
- DB : oracle (11g) <br/>
- Tool : eclipse (jdk-8u202) <br/>
- 외부 라이브러리 : jdbc(ojdbc8.jar) / jfreechart(jcommon-1.0.23.jar, jfreechart-1.0.19.jar)
